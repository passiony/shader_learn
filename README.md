# Shader教程
本工程主要是为了讲解ShaderForge插件，但是学习Shader需要很多语法基础，所以从表皮着色器讲到了顶点和片元着色器，然后再讲了shaderforge插件。

### 环境配置
1. unity5.6.6+
2. ShaderForge插件 v1.38
3. PostProcessing插件

### 博客教程
地址：https://www.jianshu.com/p/97b336cbc159
